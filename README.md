# SPPU-CGPAPercentify

Welcome to SPPU-CGPAPercentify, a Streamlit web application designed to simplify the process of converting Cumulative Grade Point Average (CGPA) to percentage for students of Savitribai Phule Pune University (SPPU). This app provides an intuitive interface to quickly calculate and visualize the percentage equivalent of CGPA based on the SPPU grading system.

## Features

- Convert CGPA to percentage with grade prediction.
- Understand the formula behind the conversion for each grade.
- Learn about SPPU CGPA to percentage conversion guidelines.
- Explore semester-wise CGPA and percentage calculations.

## How to Use

1. Enter your CGPA in the 'CGPA to Percentage' section.
2. The app will determine your grade and calculate the corresponding percentage.
3. Predict your future grades and percentages using the 'Grade Prediction' feature.
4. Calculate cumulative percentage and CGPA for multiple semesters.
5. Learn more about the app and its functionalities in the 'About' section.

## Usage Instructions

1. Clone the repository and add your Streamlit code.
2. Customize the app's content and styling as needed.
3. Deploy the app to GitHub Pages or other hosting platforms.
